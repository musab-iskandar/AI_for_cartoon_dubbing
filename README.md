# الذكاء الاصطناعي التوليدي في دبلجة الرسوم المتحركة 🎙️✨
هذا المستودع (Repository) يحتوي على مخرجات المشروع البحثي لتكليف مقرر التعلم العميق 2 (Deep Learning II) في جامعة أم القرى، والذي يستكشف دور الذكاء الاصطناعي والذكاء الاصطناعي التوليدي (Generative AI) في إحداث ثورة داخل قطاع الإعلام والترفيه، وتحديداً في توطين ودبلجة الرسوم المتحركة.

# 🎯 عن المشروع
يناقش البحث كيف يمكن تجاوز الترجمة الحرفية والدبلجة التقليدية الجامدة، واستخدام نماذج التعلم العميق لتقديم محتوى ترفيهي للأطفال بلهجاتهم المحلية (مثل النجدية، الحجازية، وغيرها)، مع الحفاظ التام على "البصمة الصوتية" والأداء العاطفي للممثل الأصلي.

# 📁 محتويات المستودع
يحتوي هذا المستودع على الملفات التالية:
- 📄 Report.pdf: التقرير البحثي الشامل (يتضمن الإطار النظري، المقارنات، الحالات الدراسية للشركات، والتحديات).
- 🌐 index.html: واجهة ويب تفاعلية مبنية باستخدام (HTML/TailwindCSS/Chart.js) لتلخيص واستعراض نتائج البحث بصرياً.
- 📝 prompt_log.md: سجل يوثق الموجهات (Prompts) التي تم استخدامها مع أدوات الذكاء الاصطناعي خلال مراحل بناء البحث وتصميم الواجهة، مع تقييم للنتائج.
